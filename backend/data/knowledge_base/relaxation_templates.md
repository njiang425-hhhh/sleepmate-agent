# 放松模板

放松模板是一套结构化的睡前放松流程。每个模板包含多个步骤，按照特定顺序执行，帮助身心从活跃状态逐步过渡到睡眠状态。

## 10 分钟快速放松

适用于时间有限但需要快速放松的情况。结合呼吸和简单身体放松。

步骤：
1. 准备就位（1分钟）：躺好，调整到最舒适的姿势，闭上眼睛。
2. 深呼吸（3分钟）：用鼻子吸气4秒，用嘴呼气6秒。重复8次。每次呼气时默念"放松"。
3. 面部放松（2分钟）：轻轻皱眉保持5秒，然后放松。紧闭双眼保持5秒，然后放松。咬紧牙关保持5秒，然后放松。让整个面部完全松弛。
4. 身体沉重（2分钟）：想象身体越来越沉重，像一块石头慢慢沉入床垫。从脚趾开始，感受沉重感向上蔓延。
5. 自然入睡（2分钟）：保持自然呼吸，不再控制任何东西。让睡意自然降临。

## 15 分钟深度放松

适用于需要更充分放松的情况。包含渐进式肌肉放松和呼吸练习。

步骤：
1. 准备就位（1分钟）：找一个安静的环境，躺好闭眼。
2. 渐进式肌肉放松（7分钟）：从双脚开始，收紧肌肉5秒然后放松。依次向上：脚趾、小腿、大腿、臀部、腹部、双手、手臂、肩膀、面部。每组肌肉配合一次呼吸。
3. 腹式呼吸（3分钟）：一只手放胸口，一只手放腹部。吸气时感受腹部上升，呼气时感受腹部下降。保持节奏平稳。
4. 身体扫描（3分钟）：将注意力从头顶缓慢移向脚底，觉察每个部位的感受。遇到紧张的部位，想象温暖的光芒包裹它。
5. 安心入睡（1分钟）：对自己说"我现在是安全的，可以安心入睡"。让呼吸自然流动。

## 20 分钟全面放松

适用于压力较大或入睡严重困难的情况。结合多种放松技术。

步骤：
1. 环境准备（2分钟）：确保房间黑暗安静，温度适宜。躺好闭眼，做三次深呼吸。
2. 渐进式肌肉放松（8分钟）：系统地收紧和放松全身肌肉群。每个部位保持紧张5-7秒，然后彻底放松15-20秒。注意放松时的感觉。
3. 呼吸练习（4分钟）：使用4-7-8呼吸法。吸气4秒，屏息7秒，呼气8秒。重复4个循环。
4. 正念觉察（3分钟）：不做任何控制，只是觉察呼吸、身体感觉和思绪。不评判，不追随，只是观察。
5. 意象放松（3分钟）：想象自己在一个宁静的场景中。调动所有感官去体验。让放松感从头顶流到脚底。

## 高压力日特别放松

专为压力水平较高（7/10以上）的日子设计。加强了肌肉放松和呼吸深度。

步骤：
1. 安全空间建立（2分钟）：想象一个让你感到完全安全和放松的空间。可以是真实存在的地方或想象的。感受这个空间的安全感。
2. 深度呼吸（3分钟）：使用方块呼吸法。吸气4秒，屏息4秒，呼气4秒，屏息4秒。专注于呼吸的节奏。
3. 全身紧张释放（5分钟）：双手握拳保持7秒，然后突然松开，感受紧张流出。双脚趾蜷缩保持7秒，然后放松。皱眉保持7秒，然后放松。咬牙保持7秒，然后放松。耸肩靠近耳朵保持7秒，然后放下。每个动作配合深呼吸。
4. 渐进式放松（5分钟）：从头到脚系统放松。在每个部位停留更长时间，感受紧张完全释放。
5. 慈悲收尾（3分钟）：对自己说"我已经做得够好了，现在可以休息了"。感受自我接纳和温暖。让身体自然沉重，进入睡眠。
