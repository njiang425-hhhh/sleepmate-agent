# 正念冥想脚本

正念冥想是通过有意识地将注意力集中在当下，不评判地观察自己的思绪和感受。规律的正念练习可以减少入睡前的思绪纷飞，降低焦虑水平。

## 呼吸觉察冥想

这是最基础的正念练习。适合冥想初学者和入睡前使用。

引导语：找一个舒适的姿势坐好或躺下。轻轻闭上眼睛。将注意力放在呼吸上。不需要改变呼吸的节奏，只是观察它。注意空气从鼻腔进入的感觉，感受胸腔和腹部的起伏。当思绪飘走时，轻轻将注意力拉回呼吸。不要责备自己走神，这是正常的。继续观察呼吸，让每一次呼吸都带你回到当下。练习5-10分钟。

## 身体扫描冥想

身体扫描是系统地将注意力移过全身各个部位的练习。它帮助识别和释放身体中存储的紧张。

引导语：躺好后闭上眼睛。将注意力放在头顶。感受那里的感觉——温暖、凉爽、紧绷或放松。不需要改变任何感觉，只是觉察。慢慢将注意力移向额头、眼睛、脸颊、下巴。注意每个部位的感受。继续向下：脖子、肩膀、上臂、前臂、双手。然后是胸部、腹部、腰部。接着是臀部、大腿、膝盖、小腿、脚踝、双脚。如果某个部位感到紧张，想象温暖的光芒包裹它，随着呼吸慢慢释放紧张。整个过程持续10-15分钟。

## 五感觉察冥想

这种方法通过调动五种感官来 grounding，帮助从纷乱的思绪中回到当下。特别适合焦虑时使用。

引导语：现在，注意你能看到的5样东西——可以是天花板的纹路、窗帘的颜色、手指的形状。然后，注意你能听到的4种声音——可能是远处的车声、自己的呼吸声、时钟的滴答声。接下来，注意你能触碰到的3种感觉——衣服接触皮肤的感觉、床铺的支撑、空气的温度。然后，注意你能闻到的2种气味——枕巾的味道、空气中的气息。最后，注意你能尝到的1种味道——嘴里残留的味道。你现在回到了当下。

## 慈悲冥想

慈悲冥想通过培养对自己和他人的善意来缓解情绪压力。它特别适合在情绪低落或自我批评时练习。

引导语：闭上眼睛，将手放在心口。对自己说："愿我平安。愿我快乐。愿我健康。愿我自在。"感受这些话语的温暖。然后，想到一个你关心的人，对他们说同样的话："愿你平安。愿你快乐。愿你健康。愿你自在。"最后，将这份善意扩展到所有人："愿所有人平安。愿所有人快乐。愿所有人健康。愿所有人自在。"让这份温暖充满全身。

## 放松意象冥想

通过想象宁静的场景来引导身心进入放松状态。可以选择任何让你感到平静的场景。

引导语：闭上眼睛，想象自己来到一个宁静的地方。可以是温暖的沙滩、安静的森林、或者任何让你感到安全的地方。用你的感官去体验这个场景——看到周围的景色，听到环境的声音，感受空气的温度和微风，闻到自然的气息。让自己完全沉浸在这个场景中。在这里，你是安全的、放松的。让这份宁静渗透到全身的每一个细胞。
